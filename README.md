# インストール

```
npm install @akimeaki/check-browser
```

# 使い方

```js
import checkBrowser from "@akimeaki/check-browser";
```

# npm公開

ビルド

```shell
npm run build
```

ログイン

```shell
npm login
```

公開

```shell
npm publish --access=public
```
